📘 About Blox

Blox is a modern, lightweight, and user-friendly blogging platform developed using the Django web framework. It is designed to help aspiring writers, developers, and content creators share their thoughts and manage posts with ease.

  • 📝 Full CRUD functionality — create, view, edit, and delete blog posts with ease.
  • 🔐 Secure authentication — includes registration, login, and logout flows with session management.
  • 🧑‍💼 Role-based access control — only the post author can edit or delete their content.
  • 📱 Fully responsive design — mobile-first layout that adapts to all screen sizes.
  • 🎨 Clean UI/UX — minimal and elegant user interface with consistent fonts and color palette.
  • 📂 Organized codebase — modular Django app structure following best practices for scalability and maintainability.
  • 🔍 Post browsing & reading — users can easily navigate and read blog content in an intuitive format.
  • 🧭 Smart navigation — dropdown menus for quick access to create post, manage posts, profile, and about page.
  • ⚙️ Built using: Django, Bootstrap 5, SQLite (or PostgreSQL), HTML5, CSS3.
  • 📌 Use Case: Ideal for personal blogging, portfolio writing, or technical documentation.

Whether you're a beginner learning Django or someone who needs a personal blog, Blox is the perfect starting point. It’s easy to deploy, clean to maintain, and open for customization.

Thank you for exploring Blox — your thoughts deserve a beautiful home. 🖋️
